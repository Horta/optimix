# optimix

Abstract function optimisation.

## Install

Enter

```bash
pip install optimix
```

from the command-line.


## Running the tests

Install dependencies

```bash
pip install -U pytest pytest-pep8
```

then run

```python
python -c "import optimix; optimix.test()"
```
